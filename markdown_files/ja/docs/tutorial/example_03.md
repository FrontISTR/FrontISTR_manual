### 解析モデル

固有値解析を対象とした実モデル検証例題の一覧を表9.3.1に示す。
また、これらのうち、EX07（タービンローダ）およびEX08（スプリング）のモデル形状を図9.3.1、図9.3.2に示す。
そのほかのモデル形状は検証内容が一致する弾性静解析用検証例題と同じ形状であり、すでに示してある。
なお、要素タイプ731および741の例題を実行するには、別途直接法ソルバーが必要である。
