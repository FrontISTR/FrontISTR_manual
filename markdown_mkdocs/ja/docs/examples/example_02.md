#### 検証例題EX02による解析性能の検証結果

検証例題EX02穴あきブロックモデルと同等のモデルを用い汎用商用ソフトABAQUS
による解析を行った。FrontISTR
と応力成分の最大値、最小値を比較した結果を図
9.2.10に示す。同図より応力成分は非常によく一致していることが分かる。

9.2.11に分割状態を示す。図
9.2.12には単一領域と8領域分割による解析結果の応力分布を示す。
